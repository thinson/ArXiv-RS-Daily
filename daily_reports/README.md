# 日报 20260310

## 📌 今日概况

今日遥感研究聚焦视觉语言模型（VLM）在地理空间任务中的应用与可靠性。SIGMAE引入光谱指数引导的多光谱MAE基础模型，GeoAlignCLIP和GeoSolver分别从细粒度对齐和过程监督角度提升VLM推理能力。CarbonBench建立碳通量零样本学习的全球基准，BuildMamba则探索Mamba架构在建筑分割与高度估计中的潜力。

## ✨ 今日亮点

- SIGMAE提出光谱指数引导的多光谱遥感MAE基础模型，融合光谱信息提升特征表示
- GeoSolver引入细粒度过程监督强化遥感VLM的链式推理能力
- CarbonBench构建碳通量零样本学习的全球基准数据集，推动空间迁移研究
